# Dhruv Shah - Personal Portfolio

A modern, animated single-page portfolio built to showcase my projects, skills, and experience. 

## Tech Stack
- **Framework:** Next.js (React 19)
- **Styling:** CSS & Tailwind Utilities
- **Animations:** Framer Motion
- **Icons:** Lucide React & React Icons

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment
This project is optimized for deployment on [Vercel](https://vercel.com). Simply link your GitHub repository to Vercel and it will automatically build and deploy your site.
